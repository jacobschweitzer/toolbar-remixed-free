Toolbar Remixed - Free.

== Description ==

There is an admin section Options -> Toolbar Remixed where you can change the options.


== Installation ==

1. Upload toolbar-remixed.zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the options under Options -> Toolbar Remixed in the Wordpress Admin section.
